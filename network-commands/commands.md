# 游깷 Network Commands

1. `ifconfig`  
   **Configure or display network interfaces**  
   游니 Forensics: View assigned IPs, detect spoofing.

2. `ping 8.8.8.8`  
   **Test network connectivity**  
   游니 Forensics: Confirm if system is online.

3. `tcpdump`  
   **Capture and analyze network traffic**  
   游니 Forensics: Catch malicious packets.

4. `ethtool eth0`  
   **Display/set Ethernet device parameters**  
   游니 Forensics: Check link status & speed.

5. `whois domain.com`  
   **Get domain registration info**  
   游니 Forensics: Investigate malicious domains.

6. `dig google.com`  
   **DNS lookup**  
   游니 Forensics: Analyze DNS spoofing.

7. `dig -x 8.8.8.8`  
   **Reverse DNS lookup**  
   游니 Forensics: Get domain from IP.

8. `nslookup domain.com`  
   **DNS resolution info**  
   游니 Forensics: Cross-verify DNS info.

9. `host domain.com`  
   **Find IP from domain or vice versa**  
   游니 Forensics: Spot domain-IP mismatches.

10. `wget http://url`  
   **Download files from the internet**  
   游니 Forensics: Grab artifacts or toolkits.
